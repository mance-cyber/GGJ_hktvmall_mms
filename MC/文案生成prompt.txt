【任務】
請「逐項處理」上面 products 陣列內的每一件產品，
為每一件產品 product 生成指定語言的電商文案，
並且用嚴格 JSON 格式輸出。

每一件產品都會包含以下欄位：
- product.productcode
- product.product_name
- product.origin
- product.weight
- product.grade

你必須根據每件產品的資料，套用下列文案要求，逐件輸出，不可以遺漏任何一個 product。

【語氣風格】
- 主要專業優雅，適量加入輕口語增加親切感（每篇不超過5-6個輕口語詞）
- 可用詞彙：超正、完全無壓力、好食到喊、上癮、穩陣、爽到飛起、絕對值得、性價比極高
- 絕對禁用：低俗/粗口/過火俚語
- 核心邏輯：痛點 → 米芝蓮酒店供應商背書 → 極致口感 → 性價比收殺

【每件產品必寫結構（繁中 / 簡中 / 英文都要遵守）】

1. 開頭直擊痛點（2-3句）：
   點出香港消費者真實煩惱：網購急凍肉/魚擔心不夠新鮮、解凍乾柴、油花不均；超市貨比不上餐廳級；想在家吃頂級日料又覺得貴又麻煩。

2. Bullet Points 硬核賣點區（7-10點，每點15-28字）：
   必須包含：
   - 至少2點：五星酒店/米芝蓮背書（如：半島酒店、文華東方、四季酒店、Zuma、Amber、Sushi Shikon）
   - 產品對應的產地/牧場/飼養規格（根據該 product 的 origin、grade 及你可合理推斷的資訊）
   - 鎖鮮技術：-18℃專業冷鏈、柴灣自有加工中心、每日空運/海運系統
   - 便利性：解凍零分別、已精準分切、10-15分鐘上桌、完全無壓力
   - 1-2點：「多位五星酒店鐵板燒/壽司主廚私下搶著用」「性價比高到癮上身」
   
   繁中用 ▸ 開頭，英文用 • 開頭

3. 極致感官描寫段落（3-5句）：
   至少用6個高端+口語混合詞：
   入口即溶、油花綻放如大理石紋、肉汁爆棚、鮮甜到震驚、堅果香尾韻超正、表面焦香內裡粉嫩、啖啖肉香、甜美餘韻纏繞、煙韌有嚼勁、真的食到喊

4. 結尾性價比收殺（2-3句）：
   必寫：「平時去米芝蓮餐廳食同級食材動輒$800-$1,200一客，而家屋企只需1/5價格就食到」
   限時誘因：「限時88折＋滿$688免運」「上週一分鐘售罄，今次真係要手快」

【欄位說明】
對於每一件產品，你都要輸出以下結構：

- short_description：Bullet Points 區（7-10點，換行用 \n 分隔）
- long_description：完整文案流（痛點開場 → 感官描寫 → 性價比收殺，段落間用 \n\n 分隔，絕對不可出現「第1段」「第2段」等標記）
- seo_title：產品名 + 核心賣點 + GogoJap（50字內）
- seo_keywords：5-8個關鍵字，逗號分隔
- target_audience：目標客群描述（30字內）

【CRITICAL - JSON 輸出規則】
1. 絕對不可包含 ```json 或 ``` 標記
2. 絕對不可有任何 JSON 以外的文字
3. 換行用 \n，段落間用 \n\n
4. 引號內容必須正確轉義
5. 直接輸出純 JSON
6. 只輸出被要求的語言，不要輸出未被要求的語言
