# 抓取功能前端使用指南

> 非技術人員也能輕鬆掌握的競爭對手監控教學

---

## 📱 功能介紹

### 這個系統能做什麼？

**自動抓取競爭對手商品資訊**
- 🏷️ 商品名稱和描述
- 💰 價格（現價 + 原價）
- 📊 折扣百分比
- 📦 庫存狀態
- ⭐ 評分和評論數
- 🖼️ 商品圖片

**自動監控價格變動**
- 📉 價格下降通知
- 📈 價格上升警告
- 📭 缺貨/補貨提醒

**生成分析報告**
- 📈 價格趨勢圖表
- 🔄 競品價格對比
- 📊 市場競爭力分析

---

## 🎯 使用場景

### 場景 1：每日價格檢查
**角色**：電商運營經理
**需求**：每天早上查看競爭對手是否調價

**操作流程**：
1. 打開系統儀表板
2. 查看「價格變動警報」卡片
3. 看到紅色標記：競爭對手 A 降價 15%
4. 點擊查看詳情，決定是否跟進調價

---

### 場景 2：新增監控商品
**角色**：商品經理
**需求**：監控新推出的競品

**操作流程**：
1. 進入「競爭對手管理」頁面
2. 選擇目標競爭對手（如：樂天日本）
3. 點擊「新增監控商品」
4. 貼上競品 URL
5. 系統自動抓取並顯示商品資訊
6. 確認無誤後儲存

---

### 場景 3：生成競爭分析報告
**角色**：策略分析師
**需求**：每週向老闆匯報市場競爭情況

**操作流程**：
1. 進入「分析報告」頁面
2. 選擇時間範圍（如：最近 7 天）
3. 選擇要對比的競爭對手
4. 點擊「生成報告」
5. 系統自動生成包含圖表的 PDF
6. 下載或直接分享

### 二、手動觸發抓取

#### 使用時機
- ✅ 發現競爭對手促銷活動開始
- ✅ 想立即更新價格數據
- ✅ 自動抓取失敗需要重試

#### 操作步驟

**方式 A：單個商品**
1. 進入商品詳情頁面
2. 點擊右上角「🔄 立即更新」按鈕
3. 看到轉圈圈動畫（約 5-10 秒）
4. 顯示「✓ 更新完成」

**方式 B：整個競爭對手**
1. 在競爭對手管理頁面
2. 找到目標競爭對手
3. 點擊「立即抓取」按鈕
4. 看到進度條（抓取所有商品需要 1-5 分鐘）
5. 完成後顯示「✓ 已更新 15/15 個商品」

**方式 C：全部競爭對手**
1. 在主儀表板
2. 點擊「全部更新」按鈕
3. 系統會提示：
   ```
   這將抓取所有競爭對手的 53 個商品
   預計需要 5-10 分鐘
   確定要繼續嗎？

   [取消]  [確定]
   ```
4. 確認後在背景執行
5. 完成時發送通知

#### 警報處理流程

**收到警報後的決策樹**：
```
收到價格下降警報
    ↓
查看競爭對手新價格
    ↓
對比我方價格
    ↓
    ├─→ 我方更便宜 → 無需行動
    │
    ├─→ 差距小於 5% → 觀察幾天
    │
    └─→ 差距大於 5% → 考慮調價
            ↓
        分析利潤空間
            ↓
            ├─→ 可降價 → 調整價格
            │
            └─→ 不可降價 → 增加行銷預算
```

### 價格趨勢圖解讀

```
價格 ($)
10,000 ┤

 9,000 ┤   ╱      ╲
 8,500 ┤  ●        ●──●
 8,000 ┤          ╱    ╲
 7,500 ┤    ○────○      ○───○  [競爭對手 A]
 7,000 ┼────────────────────────
       12/1  12/8  12/15  12/22
```

**看圖重點**：
1. **趨勢方向**
   - 向下 = 降價促銷
   - 向上 = 恢復原價
   - 平穩 = 價格穩定

2. **價差**
   - 兩線距離 = 價格差距
   - 距離擴大 = 競爭力變化
   - 距離縮小 = 價格趨同

3. **波動性**
   - 頻繁上下 = 促銷活躍
   - 平穩線條 = 價格策略保守
